# Apple HIG - Spacing & Layout

## iOS/iPadOS Spacing

### Standard Spacing
- **4pt**: Minimal spacing
- **8pt**: Standard unit (base)
- **12pt**: Small spacing
- **16pt**: Medium spacing
- **20pt**: Large spacing
- **24pt**: Extra large spacing
- **32pt**: Section spacing
- **44pt**: Minimum touch target

### Safe Area Insets
- **Top**: Status bar + notch/Dynamic Island
- **Bottom**: Home indicator area
- **Sides**: Screen edges

### Margins
- **Standard**: 16pt (iPhone), 20pt (iPad)
- **Reading Width**: 672pt maximum for text

## Touch Targets
- **Minimum**: 44pt × 44pt
- **Recommended**: 48pt × 48pt or larger
- **Spacing between**: 8pt minimum

## Layout Guides

### iPhone
- **Compact Width**: Portrait mode
- **Regular Height**: Portrait mode
- **Compact Height**: Landscape mode (except Plus/Max)

### iPad
- **Regular Width**: All orientations
- **Regular Height**: All orientations
- **Multitasking**: 1/3, 1/2, 2/3 splits

## Component Spacing

### Navigation Bar
- Height: 44pt (compact), 96pt (large title)
- Title: 34pt (large), 17pt (standard)
- Padding: 16pt horizontal

### Tab Bar
- Height: 49pt (iPhone), 50pt (iPad)
- Icon size: 25pt × 25pt (iPhone), 30pt × 30pt (iPad)

### Table View
- Row height: 44pt (default), 56pt (subtitle)
- Cell padding: 16pt horizontal
- Section spacing: 35pt

### Buttons
- Padding: 12pt horizontal, 8pt vertical (minimum)
- Corner radius: 8pt (small), 10pt (medium), 12pt (large)

## Typography Spacing
- Line spacing: 1.2-1.5 for body text
- Paragraph spacing: 16pt
- Heading spacing: 24pt

## macOS Spacing

### Window Margins
- **Standard**: 20pt
- **Content**: 16pt padding

### Toolbar
- Height: 52pt
- Item spacing: 8pt

### Sidebar
- Width: 200-300pt
- Item height: 24pt
- Padding: 8pt
