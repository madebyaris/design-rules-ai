# Adobe Spectrum - Spacing Scale

## Overview
Spectrum uses an 8px base grid system for consistent spacing throughout the UI.

## Spacing Scale

### Base Units (8px grid)
- **0**: 0px
- **1**: 4px (half unit - rare, for fine adjustments)
- **2**: 8px (base unit)
- **3**: 12px
- **4**: 16px
- **5**: 20px
- **6**: 24px
- **8**: 32px
- **10**: 40px
- **12**: 48px
- **16**: 64px
- **20**: 80px
- **24**: 96px
- **32**: 128px

## Usage Guidelines

### Component Spacing
- **Padding (small components)**: 8-16px
- **Padding (medium components)**: 16-24px
- **Padding (large components)**: 24-32px
- **Gap between elements**: 8px, 16px, 24px

### Layout Spacing
- **Section padding**: 32px, 48px, 64px
- **Container max-width**: 1280px
- **Column gaps**: 16px, 24px, 32px

### Vertical Rhythm
- **Paragraph spacing**: 16px
- **Heading spacing**: 24px, 32px
- **Section spacing**: 48px, 64px, 96px

## Responsive Adjustments
- **Mobile (< 768px)**: Reduce spacing by 25-50%
- **Tablet (768-1024px)**: Standard spacing
- **Desktop (> 1024px)**: Standard or increased spacing

## Component-Specific Spacing

### Buttons
- Padding: 8px 16px (small), 12px 24px (medium), 16px 32px (large)
- Gap between buttons: 8px

### Forms
- Label to input: 8px
- Input to input: 16px
- Field group spacing: 24px

### Cards
- Inner padding: 16px, 24px
- Gap between cards: 16px, 24px

### Navigation
- Nav item padding: 8px 16px
- Nav item gap: 4px, 8px
