# Adobe Spectrum - Typography

## Overview
Spectrum uses Adobe Clean (proprietary) or system fonts as fallbacks with a clear type scale.

## Font Families
- **Primary**: Adobe Clean, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **Monospace**: "Source Code Pro", Monaco, monospace

## Type Scale

### Heading Sizes
- **Heading 1**: 36px / 2.25rem (line-height: 1.2)
- **Heading 2**: 28px / 1.75rem (line-height: 1.3)
- **Heading 3**: 22px / 1.375rem (line-height: 1.4)
- **Heading 4**: 18px / 1.125rem (line-height: 1.4)
- **Heading 5**: 16px / 1rem (line-height: 1.5)
- **Heading 6**: 14px / 0.875rem (line-height: 1.5)

### Body Sizes
- **Body Large**: 18px / 1.125rem (line-height: 1.6)
- **Body**: 14px / 0.875rem (line-height: 1.6)
- **Body Small**: 12px / 0.75rem (line-height: 1.5)
- **Body XSmall**: 11px / 0.6875rem (line-height: 1.5)

### Detail/Code Sizes
- **Detail**: 12px / 0.75rem (line-height: 1.5)
- **Code**: 13px / 0.8125rem (line-height: 1.6)

## Font Weights
- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **Bold**: 700
- **Heavy**: 900 (rare, for emphasis)

## Usage Guidelines

### Headings
- Use sentence case for most headings
- Use bold weight for H1-H3
- Use medium weight for H4-H6
- Limit line length to 60-70 characters

### Body Text
- Use regular weight for body text
- Line height: 1.5-1.6 for readability
- Paragraph spacing: 1em
- Max width: 65-75 characters per line

### UI Text
- **Buttons**: Medium weight, 14px
- **Labels**: Regular weight, 12-14px
- **Help text**: Regular weight, 12px
- **Error text**: Regular weight, 12px, red color

## Responsive Typography
- **Mobile**: Reduce heading sizes by 10-20%
- **Tablet**: Standard sizes
- **Desktop**: Standard or slightly increased

## Accessibility
- Minimum font size: 12px (preferably 14px)
- Contrast ratio: 4.5:1 for body text, 3:1 for large text (18px+)
- Line height: minimum 1.5 for body text
- Letter spacing: normal (avoid tight spacing)
