## Font Weights
- **Regular**: 400
- **Semibold**: 600
- **Bold**: 700 (rare, for emphasis)

## Variable Font Axes
Segoe UI Variable supports:
- **Weight**: 400-700
- **Optical Size**: Automatic adjustment

## Usage Guidelines

### Hierarchy
1. Use Title for page/section headers
2. Use Subtitle for subsections
3. Use Body for content
4. Use Caption for metadata, labels

### Emphasis
- **Strong**: Use semibold (600) weight
- **Subtle**: Use secondary text color
- **Italic**: Use sparingly for emphasis

### Line Length
- Optimal: 50-75 characters
- Maximum: 90 characters
- Use max-width to constrain

## Accessibility
- Minimum size: 12px (preferably 14px)
- Body text contrast: 4.5:1
- Large text (18px+): 3:1
- Line height: 1.4-1.5 for body
- Letter spacing: Normal (avoid tight)

## Responsive Typography
- Use relative units (rem, em)
- Scale down on mobile (10-20%)
- Maintain readability at all sizes
- Test with browser zoom (200%)

## Platform Variations

### Windows
- Segoe UI Variable (Windows 11)
- Segoe UI (Windows 10)
- ClearType rendering

### Web
- System font stack
- -apple-system, BlinkMacSystemFont
- Segoe UI, Roboto, sans-serif

### Mobile
- System fonts (SF Pro, Roboto)
- Larger touch targets
- Increased line height
