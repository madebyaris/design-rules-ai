# Microsoft Fluent 2 - Spacing & Layout

## Spacing Scale (4px base)
- **2px**: Minimal spacing (rare)
- **4px**: Tight spacing
- **8px**: Small spacing
- **12px**: Medium-small spacing
- **16px**: Medium spacing (base unit)
- **20px**: Medium-large spacing
- **24px**: Large spacing
- **28px**: Extra large spacing
- **32px**: Section spacing
- **36px**: Large section spacing
- **40px**: Extra large section
- **48px**: Maximum spacing

## Density Levels

### Compact
- Reduced padding and spacing
- For data-dense interfaces
- Desktop applications

### Normal
- Standard spacing
- Default for most interfaces
- Balanced density

### Spacious
- Increased padding and spacing
- Touch-friendly
- Mobile and tablet

## Component Spacing

### Buttons
- **Small**: 20px height, 8px horizontal padding
- **Medium**: 32px height, 12px horizontal padding
- **Large**: 40px height, 16px horizontal padding
- Gap between: 8px

### Inputs
- Height: 32px (default)
- Padding: 8px horizontal
- Label spacing: 4px above

### Cards
- Padding: 12px (compact), 16px (normal), 20px (spacious)
- Gap between cards: 8px, 12px, 16px

### Lists
- Item height: 32px (compact), 44px (normal), 56px (spacious)
- Item padding: 12px horizontal
- Gap between items: 0px (compact), 2px (normal), 4px (spacious)

## Layout Grid

### Breakpoints
- **Small**: 0-639px (mobile)
- **Medium**: 640-1023px (tablet)
- **Large**: 1024-1365px (desktop)
- **XLarge**: 1366-1919px (large desktop)
- **XXLarge**: 1920px+ (ultra-wide)

### Columns
- **Mobile**: 4 columns
- **Tablet**: 8 columns
- **Desktop**: 12 columns

### Gutters
- 16px (mobile)
- 24px (tablet)
- 24px (desktop)

### Margins
- 16px (mobile)
- 32px (tablet)
- 40px (desktop)

## Touch Targets
- Minimum: 32px × 32px (mouse)
- Minimum: 40px × 40px (touch)
- Recommended: 44px × 44px (touch)
- Spacing between: 8px minimum

## Elevation Spacing
Shadows create perceived depth:
- Level 2: 2px
- Level 4: 4px
- Level 8: 8px
- Level 16: 16px
- Level 64: 64px (maximum)
