# IBM Carbon - Typography

## Overview
Carbon uses IBM Plex (open-source) as its primary typeface with a systematic type scale.

## Font Families
- **Sans**: 'IBM Plex Sans', 'Helvetica Neue', Arial, sans-serif
- **Serif**: 'IBM Plex Serif', 'Georgia', Times, serif
- **Mono**: 'IBM Plex Mono', 'Menlo', 'DejaVu Sans Mono', monospace

## Usage Guidelines

### Productive Type (Default)
Use for:
- Data tables
- Forms
- Dashboards
- Application UI
- Dense information displays

### Expressive Type
Use for:
- Marketing pages
- Landing pages
- Editorial content
- Large displays
- Storytelling

## Accessibility
- Minimum font size: 12px
- Body text contrast: 4.5:1 minimum
- Large text (18px+): 3:1 minimum
- Line height: 1.4-1.5 for body text
- Paragraph max-width: 66 characters

## Responsive Typography
- Use rem units for scalability
- Maintain type scale ratios across breakpoints
- Reduce display sizes by 20-30% on mobile
- Keep body text readable (minimum 14px on mobile)
