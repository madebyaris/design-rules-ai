# Material Design 3 - Spacing & Layout

## Spacing Scale (4dp base)
- **4dp**: Minimal spacing
- **8dp**: Standard spacing unit
- **12dp**: Small spacing
- **16dp**: Medium spacing
- **24dp**: Large spacing
- **32dp**: Extra large spacing
- **48dp**: Section spacing
- **64dp**: Large section spacing

## Layout Grid
- **Columns**: 4 (mobile), 8 (tablet), 12 (desktop)
- **Gutters**: 16dp (mobile), 24dp (tablet), 24dp (desktop)
- **Margins**: 16dp (mobile), 24dp (tablet), 24dp (desktop)

## Component Spacing

### Buttons
- Padding: 24dp horizontal, 10dp vertical (medium)
- Gap: 8dp between buttons
- Icon spacing: 8dp from text

### Cards
- Padding: 16dp
- Content spacing: 8-16dp

### Lists
- Item height: 56dp (one-line), 72dp (two-line), 88dp (three-line)
- Item padding: 16dp horizontal
- Icon to text: 16dp

### App Bars
- Height: 64dp (top), 80dp (bottom)
- Padding: 16dp horizontal
- Icon size: 24dp

## Responsive Breakpoints
- **Compact**: 0-599dp (phone portrait)
- **Medium**: 600-839dp (tablet, phone landscape)
- **Expanded**: 840dp+ (desktop, large tablet)
