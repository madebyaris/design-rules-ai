Card list — skeleton loader

Skeleton rules

  • Match the shape of the eventual content (rectangles for text, circles for avatars).
  • Use only when load > 200ms — anything faster, skip the skeleton (it adds perceived latency).
  • Honor prefers-reduced-motion — disable shimmer, leave a static placeholder.
  • Never use skeletons inside buttons. Use a spinner inside the button instead.
  • Cap skeleton rows to what fits the visible viewport — don't render 50 fake rows.
